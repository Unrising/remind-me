Project Overview

"Remind Me" is a personal task management tool that helps users stay organized by setting reminders, managing deadlines, and visualizing timelines.

Description

This project is built using angular 17 (standalone) and aims to provide a clean, modern interface for task creation and visualisation.

Technologies Used

Here's a preview of the application UI in various status - Including the home screen, calendar view and task manager.

Technologie Used

  • Angular 17 (Standalone Components)
  • TypeScript
  • CSS

Objectives

- Learn and apply Angular standalone architecture
- Practice component-based design
- Build a responsive layout
